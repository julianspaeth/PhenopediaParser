# PhenopediaParser

## Getting Started

### Install

```pip install phenopedia_parser```

### Usage

```
from phenopedia_parser.phenopedia_parser import PhenopediaParser

pheno_dataframe = PhenopediaParser.parse(search_term='lung', dataframe=True)
print(pheno_dataframe)
```
